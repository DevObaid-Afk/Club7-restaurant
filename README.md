# 🍽️ CLUB-7 Restaurant Website

A modern, high-end restaurant website concept designed for **CLUB-7 FAMILY RESTAURANT & BAR (Nalasopara, Mumbai)**.

This project focuses on creating a **premium digital experience** that reflects the restaurant’s real-world vibe — combining luxury dining, nightlife energy, and seamless user interaction.

---

## 🚀 Features

- 🎨 Dark luxury UI (neon + glassmorphism)
- 📱 Fully responsive (mobile-first)
- ⚡ Smooth animations & micro-interactions
- 🎥 Immersive hero section
- 🍛 Interactive menu showcase
- 📅 Table reservation system (UI)
- ⭐ Customer reviews section
- 📍 Location & contact integration

---

## 🧩 Tech Stack

- **Frontend:** React.js / Next.js  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion / GSAP  
- **3D Elements:** Three.js  

---

## 🎯 Purpose

This project demonstrates how a modern website can:
- Enhance brand identity  
- Attract more customers  
- Increase direct reservations  
- Create a premium online presence
